This is an example of how to perform mathematical operations on dates. The most important part is to convert the date to milliseconds.

new Date(d)?.setDate(new Date(d)?.getDate()) where d is the date in 'yyyy-mm-dd' format

Once converted to milliseconds, you can perform mathematical operations and convert the result back to years, months, days, hours, or whatever is needed.

Here we want to know how many days in the past a specific day is in relation to today. Using the code above set d to the current date and store in a variable named currentDate

currentDate = new Date(d)?.setDate(new Date(d)?.getDate())

Do the same for a previous date

previousDate = new Date(d)?.setDate(new Date(d)?.getDate()) where d is a date in the past

(currentDate - previousDate)/(24*60*60*1000) Convert from milliseconds to days to get the number of days in the past
